# postfiatorg.github.io

Public website, whitepaper, benchmark pages, and machine-readable project surfaces for Post Fiat.

> Post Fiat is an XRP-derived network for capital markets and collective intelligence. It is currently live on public testnet, not production mainnet.

## Canonical project resources

- About: https://postfiat.org/about/
- Homepage: https://postfiat.org/
- Whitepaper: https://postfiat.org/whitepaper/
- Validator benchmark: https://postfiat.org/validator-benchmark/
- Task Node: https://tasknode.postfiat.org/
- Assistant summary: https://postfiat.org/llms.txt
- Machine-readable JSON summary: https://postfiat.org/postfiat-project.json

- The public Post Fiat site
- The whitepaper and benchmark pages
- Assistant-facing summary files such as `llms.txt`
- Homepage and landing-page implementations
