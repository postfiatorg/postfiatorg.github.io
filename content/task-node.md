The Task Node is the current contributor surface for Post Fiat. Use it to create or link a wallet, receive tasks, submit evidence, track rewards, and maintain your public network profile.

## What You Can Do

- Log in with a supported identity provider.
- Link and unlock a wallet for Task Node activity.
- Request, accept, decline, and submit tasks.
- Track outstanding tasks, verification status, rewarded work, and in-flight PFT.
- Update your profile, context, and public contributor surface.

## Setup Notes

New contributors should start in the Task Node app. If you need wallet setup help, use the wallet guide linked from this page and then return to the app to validate your wallet.

The legacy PFTasks app is no longer the live task system. It remains available only as a temporary fallback for seed backup and direct wallet sends during the cutover window.
