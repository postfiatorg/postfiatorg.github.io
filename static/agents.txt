## Current Validator Setup Summary

Default target is a fresh Post Fiat testnet validator using Docker Compose and the official light validator image.

- Network: `testnet`
- Compose file: `https://raw.githubusercontent.com/postfiatorg/postfiatd/main/scripts/docker-compose-external-validator.yml`
- Default image from compose: `agtipft/postfiatd:testnet-light-latest`
- Current recommended explicit tag as of 2026-05-01: `agtipft/postfiatd:testnet-light-1.0.4`
- Public peer port: TCP `2559`
- Admin ports should remain firewalled from the public internet: `5005`, `6005`, `6006`, `50051`

Do not request or expose validator private keys or validator tokens. Treat `validator-keys.json` and `[validator_token]` material as secrets.
