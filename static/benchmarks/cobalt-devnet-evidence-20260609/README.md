# Cobalt Devnet Evidence Bundle

Created: 2026-06-09

This is the public evidence bundle cited by the Cobalt implementation-evidence
draft. It mirrors the JSON reports cited in the article and adds a compact
implementation-surface inventory.

The bundle is intentionally not a full source-code mirror. It excludes raw logs,
private operator material, credentials, local placement inputs, and unrelated
working-tree state. Report-level `git` fields remain the authority for the
clean revisions used by the release, replay, and readiness gates.

Use `manifest.json` for file sizes, SHA-256 checksums, and roles.
